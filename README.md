Yuuma Portfolio Advanced

HTML / CSS / JavaScript を使用して制作した、実務レベルを想定したフロントエンドポートフォリオです。
アクセシビリティ・UI状態管理・フォーム処理・データ保存など、実務で必要となる機能を含めて設計・実装しています。

URL

https://yuuma-dev1.github.io/yuuma-portfolio-advanced/

使用技術

・HTML5
・CSS3（カスタムプロパティ / レスポンシブ対応）
・JavaScript（Vanilla JS）
・localStorage
・IntersectionObserver
・GitHub Pages（公開）

実装機能
レスポンシブナビゲーション

・スマートフォン表示でハンバーガーメニューを表示
・aria-expanded によるアクセシブルな状態管理
・Escapeキーでメニューを閉じる処理

スクロールアニメーション

・IntersectionObserver を使用
・要素が画面に入るとフェードイン表示

ダークモード切替

・ボタンでテーマ切替
・localStorage に状態を保存
・再訪問時にテーマを復元

プロジェクトのフィルタ・検索

・カテゴリで絞り込み
・キーワード検索
・検索状態を localStorage に保存

お問い合わせフォーム（デモ）

・未入力チェック
・メール形式チェック
・送信中 / 成功状態の表示
・JavaScript による状態管理

制作目的

フロントエンドエンジニアとしての就職を目的とし、
実務で求められるHTML構造設計・CSS設計・JavaScriptによるUI制御のスキルを証明するために制作しました。
